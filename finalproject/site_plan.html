News Time Website Plan

Site Name

Name: News Time

This name represents a global news hub delivering world events and trending topics in real time using free APIs. The name was chosen because it is simple, direct, and conveys immediacy. Optional Domain: newstime.org

Site Purpose

The purpose of News Time is to deliver curated, real-time news from multiple free APIs, offering content in categories like world events, politics, technology, sports, and entertainment. The site will serve as a one-stop destination for people who want to stay informed about global events without navigating multiple platforms.

Scenarios

  • “What are the top global headlines right now?”
  • “Can I filter news by category such as technology or sports?”

Color Schema

Primary Color: #1a73e8 (Blue) – Used for headings, links, and accents.
Accent Color: #fbbc04 (Yellow) – Used for highlights, borders, and call-to-action buttons.
Background Color: #f4f4f4 – For page background.
Text Color: #333 – Standard text.

Typography

Primary Font: Roboto (Google Font)
Usage: - Headings (bold weight) - Body text (regular weight)
